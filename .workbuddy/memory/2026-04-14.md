// ❌ 错误的导入方式

const audioBuffer = await tts(text, { voice: voiceId, rate, volume });

- 直接调用函数：`tts(text, options)` 返回 Buffer

router.use(express.json());

- 方法3：逐字符向后查找完整 JSON
