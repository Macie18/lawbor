### 12. 福利政策指南回到顶部按钮修复（最终版）
- **问题**：按钮一直不显示，即使 `showBackToTop` 状态正确变化
- **根本原因**：
  - 页面使用了 Framer Motion (`motion.div`)
  - 父元素可能有 `transform`、`filter` 或 `perspective` 属性
  - 这些属性会导致 `position: fixed` 相对于父元素而非视口定位
- **解决方案**：
  - 使用 `createPortal` 将按钮渲染到 `document.body`
  - 提升按钮的 `z-index` 到 `9999`
- **修改文件**：`src/pages/BenefitsGuide.tsx`
- **技术细节**：
  - 导入 `createPortal` from `react-dom`
  - `{createPortal(<AnimatePresence>...</AnimatePresence>, document.body)}`
- **位置调整**：放在 AI 聊天按钮左边 (`right-24`)，使用灰色背景区分

### 11. 税务计算 PDF 导出修复（最终版 v4）
- **问题**：
  1. `oklch` 颜色格式不被 html2pdf.js 支持（Tailwind v4 使用）
  2. iframe 方案仍然报错
- **最终解决方案**：临时禁用父页面的 Tailwind 样式表
  1. 创建 iframe 隔离 PDF 内容
  2. 临时禁用父页面的 Tailwind 相关样式表
  3. 临时移除包含 `oklch` 的 `<style>` 标签
  4. 生成 PDF 完成后恢复所有样式
- **修改文件**：`src/services/taxReportPdf.ts`
- **技术细节**：
  - `styleEl.disabled = true` 禁用样式表
  - `style.remove()` 移除 style 标签
  - 保存 `parentElement` 和 `nextSibling` 用于恢复
  - Promise then/catch 确保样式恢复
