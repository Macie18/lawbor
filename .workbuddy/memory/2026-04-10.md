### 问题 6：合同文本提取公司名称错误
- **错误**：提取到 `甲方（用人单位）：万得信息技术股份有限公司`
- **原因**：正则表达式不够健壮
  - `甲方[：:]\s*` 只匹配空白字符，不匹配 `（用人单位）`
  - Fallback 正则匹配到了错误的前缀
- **修复**：
  - 允许前缀和冒号之间有任意字符：`甲方[^：:\n]*[：:]`
  - 排除括号字符：`[^\s，。；,.\n（）\(\)]`
  - 过滤包含"甲方"/"乙方"的错误匹配
- **提交**：`75cc9b9 fix: 修复合同文本提取公司名称的正则表达式`

## 模拟面试语音识别网络问题修复

### 问题 7：语音识别 network 错误
- **错误**：`ERR_INTERNET_DISCONNECTED` + `Speech recognition error: network`
- **原因**：Web Speech API（Chrome）需要连接 Google 语音识别服务
- **影响**：在中国大陆，Google 服务被墙，导致语音识别无法使用
- **修复**：
  - 添加网络状态检测（`navigator.onLine` + 事件监听）
  - 在设置页面显示离线警告和网络提示
  - 明确告知用户语音识别需要国际网络连接
- **文件**：
  - `src/pages/Interview/index.tsx` - 添加 isOnline 状态和警告 UI
  - `src/contexts/TranslationContext.tsx` - 添加中英文提示文案
- **用户体验改进**：
  - 离线时显示红色警告：`网络已断开`
  - 在线时显示琥珀色提示：`语音识别需要连接国际网络（Google 服务）`

### 技术要点
- Web Speech API 是浏览器原生 API，不是后端服务
- Chrome 使用 Google 服务，需要国际网络
- Safari/Firefox 可能有不同的语音识别后端
- 代码已有 `network` 错误的翻译，但用户在使用前不知道依赖
