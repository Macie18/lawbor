### 网络连接问题排查 (23:02)
- 用户反馈 API 调用出现 `ERR_CONNECTION_RESET` 错误
- 问题诊断：
  - Supabase Anon Key 格式错误（使用了错误的 `sb_publishable_` 格式）
  - 系统代理配置在 `127.0.0.1:17890`（Clash HTTP 代理端口）
  - 终端可访问 API（返回 401），但浏览器未正确使用代理
- 解决方案：
  - 需要从 Supabase Dashboard 获取正确的 JWT 格式 Anon Key（以 `eyJ` 开头）
  - 配置浏览器使用系统代理或安装 SwitchyOmega 扩展
  - 可启用 Clash TUN 模式实现全局代理
