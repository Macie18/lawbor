### 测试用例1：月薪17500元（第1个月）

**输入**:
- 月税前工资：¥17500
- 五险一金：¥875
- 专项附加扣除：¥0
- 月份：第1个月

**计算过程**:
```
累计收入 = 17500
累计社保 = 875
累计专项附加扣除 = 0
基本减除费用 = 5000

累计应纳税所得额 = 17500 - 875 - 0 - 5000 = 11625元

适用税率 = 3%（不超过36000元）
速算扣除数 = 0

累计个税 = 11625 × 3% - 0 = 348.75元
本月个税 = 348.75 - 0 = 348.75元
```

**预期结果**:
- ✅ 本月个税：**¥348.75**
- ✅ 本月税后收入：**¥16,276.25**

**修复前错误结果**:
- ❌ 本月个税：¥952.5（错误）
- ❌ 误差：+¥603.75

### 测试用例2：月薪20000元（累计6个月）

**输入**:
- 月税前工资：¥20000
- 五险一金：¥2000
- 专项附加扣除：¥0
- 月份：第6个月

**计算过程**:
```
累计收入 = 20000 × 6 = 120000
累计社保 = 2000 × 6 = 12000
累计专项附加扣除 = 0
基本减除费用 = 5000 × 6 = 30000

累计应纳税所得额 = 120000 - 12000 - 0 - 30000 = 78000元

适用税率 = 10%（超过36000至144000元）
速算扣除数 = 2520

累计个税 = 78000 × 10% - 2520 = 5280元

前5个月已缴税额计算：
第1-5个月累计应纳税所得额 = (20000 - 2000 - 0 - 5000) × 5 = 65000元
前5个月累计个税 = 65000 × 10% - 2520 = 3980元

本月个税 = 5280 - 3980 = 1300元
```

**预期结果**:
- ✅ 本月个税：**¥1300**
- ✅ 累计个税：**¥5280**
- ✅ 本月税后收入：**¥16700**
