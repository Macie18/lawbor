### 示例计算对比

**假设**：月税前工资 ¥17500，五险一金 ¥875，专项附加扣除 ¥0，第1个月

#### 正确计算（文档标准）：
- 累计应纳税所得额 = 17500 - 875 - 5000 = **11625元**

- 本月个税 = 11625 × 3% - 0 = **¥348.75**

- 累计应纳税所得额 = 11625元
- 适用税率 = 10%（超过3000至12000元）❌
