## 🎯 使用说明

### 五险一金政策（全国通用）
1. 点击"全国通用"按钮查看通用政策
2. 或选择省份和城市查看地方政策

### 就业扶持政策（地区特定）
1. 必须选择省份
2. 如有多个城市，需进一步选择城市
3. 选择后自动加载该地区政策文件

## 🔍 数据结构

```typescript
interface City {
  id: string;        // 城市ID
  name: string;      // 城市中文名
  nameEn: string;    // 城市英文名
}

interface Province {
  id: string;        // 省份ID
  name: string;      // 省份中文名
  nameEn: string;    // 省份英文名
  cities: City[];    // 包含的城市列表
}
```

## 📌 注意事项

### 1. 政策文件命名规则
- 五险一金: `/public/policies/{cityId}.md` 或 `{cityId}-en.md`
- 就业扶持: `/public/policies/employment/{cityId}.md`

### 2. 需要补充的政策文件
目前只有4个直辖市有政策文件：
- ✅ beijing.md / beijing-en.md
- ✅ shanghai.md / shanghai-en.md
- ✅ tianjin.md / tianjin-en.md
- ✅ chongqing.md / chongqing-en.md

其他省份和城市的政策文件需要逐步补充。

### 3. 文件缺失时的提示
如果用户选择了没有政策文件的城市，会显示：
- 中文: "该地区政策文件尚未上线"
- 英文: "English policy not yet available for this city"
