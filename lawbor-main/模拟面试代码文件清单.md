## 迁移注意事项

1. **API Key 配置**：`llmService` 依赖 `VITE_DEEPSEEK_API_KEY` 环境变量，新项目需配置
2. **Lottie 依赖**：需安装 `lottie-react`，且 `public/siri.json` 必须一起迁移
3. **翻译 key 提取**：TranslationContext 中 `interview.*` 相关 key 需整体搬到新项目的翻译体系
4. **浏览器 API 兼容**：语音识别（SpeechRecognition）和语音合成（SpeechSynthesis）仅 Chrome/Edge 完整支持
5. **server.ts 可选**：当前 Interview 页面已完全走前端直连 LLM API，WebSocket 管线（server.ts + VoiceCallService + STT/TTS 服务）是旧版或备用方案，可按需迁移
6. **chat.ts 中面试 prompt**：`buildInterviewerSystemPrompt` 与 `llmService.buildSystemPrompt` 功能重叠，迁移时选其一即可
