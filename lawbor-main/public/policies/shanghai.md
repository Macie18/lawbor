# 上海市五险一金及社保相关政策汇总
**基于国家税务总局上海市税务局社保费专栏官方内容**

> 注：本汇总全部基于上海市现行有效政策，自2020年11月1日起，上海市企业职工各项社会保险费统一交由税务部门征收，所有缴费相关业务均可通过上海税务渠道办理。

## 一、五险一金

### （一）缴纳比例查询（上海市专属标准）

#### 2. 灵活就业人员缴费比例
灵活就业人员是指年满16周岁且男性未满60周岁、女性未满55周岁，在本市劳动就业的自雇人员、无雇工个体工商户、未在用人单位参加基本养老、医疗保险的非全日制从业人员以及其他灵活就业人员。
- 职工养老保险：缴费比例20%，全部由个人承担
- 职工医疗保险（含生育保险）：缴费比例10%，全部由个人承担
- 可选险种：住房公积金，可自愿缴存，缴存比例10%-24%，月缴存额全额由个人承担，缴存满6个月可享受与企业职工同等的公积金贷款待遇
- 缴费基数：可在当年上海市社保缴费基数上下限（2025.7-2026.6：上限37302元/月，下限7460元/月）之间自主选择

#### 3. 官方查询渠道
国家税务总局上海市税务局官网社保费专栏、上海市人社局官网、上海市医保局官网、上海住房公积金网。

### （二）断缴影响说明

#### 1. 五险断缴核心影响
| 险种 | 断缴具体影响 |
| :--- | :--- |
| 养老保险 | 根据《中华人民共和国社会保险法》规定，参加基本养老保险的个人，达到法定退休年龄时累计缴费满十五年的，按月领取基本养老金。累计缴费年限含视同缴费年限，不要求连续缴费，中断前后的缴费年限、个人账户储存额合并累计。<br/>养老金计发遵循“多缴多得，长缴多得”原则，中断缴费时间越长，累计缴费年限越短，个人账户累计金额越少，退休后养老金待遇相对降低；达到法定退休年龄时累计缴费不足15年的，需延长缴费至满15年。<br/>仅单位就业期间应缴未缴的社保费可由用人单位申请补缴，补缴时段距办理时间6个月内的，需提供录用聘用材料；超6个月的，需额外提供档案、工资发放凭证、法律文书等资料。 |
| 医疗保险（含生育保险） | 断缴次月起停止享受医保统筹报销待遇，个人账户余额可正常使用；灵活就业人员断缴超3个月的，重新缴费后设6个月待遇等待期；灵活就业人员生育待遇申领需生育/流产前连续缴满12个月，断缴直接影响申领资格；断缴会影响医保累计缴费年限，进而影响退休后终身医保待遇资格。 |
| 失业保险 | 缴费年限累计计算，断缴不影响已累计时长，但领取失业保险金需满足失业前累计缴费满1年的核心条件。 |
| 工伤保险 | 断缴期间发生工伤的，工伤保险基金不予支付相关待遇，全部法定待遇由用人单位全额承担。 |
| 附加影响 | 上海市购房、落户、居住证积分等政策对社保连续缴纳有严格要求，断缴会直接影响相关资格申请。 |

#### 2. 公积金断缴核心影响
- 公积金贷款申请需连续缴存满6个月（灵活就业人员12个月），断缴1个月及以上（无缝续缴除外）会导致连续缴存时长清零，无法申请公积金贷款；
- 断缴期间账户正常封存，不影响已有账户余额，符合特定条件（如外省市户籍封存满6个月）的可正常提取；
- 已发放的公积金贷款，断缴不影响正常还款，按时足额还贷即可，但需注意逐月对冲可能断档，应及时补足还款资金。

### （三）转移接续流程

#### 1. 社会保险跨省/跨统筹区转移接续
上海社保费由税务部门统一征收，转移接续核心经办主体为社保/医保经办机构，税务部门负责缴费记录核对确认，全程可网办，无需往返两地。

- **基本养老保险转移**
参保人通过全国人社政务服务平台、国家社会保险公共服务平台或掌上12333移动应用，选择“社保转移申请转入地开通地区查询”服务，查看拟转入地是否已开通网上申请服务。已开通服务的地区，参保人选择“社保转移申请”服务，经实人认证成功后，填写转移申请相关信息并提交申请。转入地经办机构负责受理申请、后台审核，经核查符合转入条件的，开启后续转移流程。

- **医疗保险转移**
  1. 申请方式：参保人员/用人单位可通过线上登录医保信息平台录入申请信息，或线下到经办机构窗口提供有效身份证件/医疗保障凭证提交转移接续申请；
  2. 资格校验：经办机构依托医保信息平台进行资格校验，符合条件的成功受理，不符合条件的不予受理并一次性告知原因；
  3. 转移办理：转出地和转入地经办机构协同办理转移接续手续，转出地10个工作日内生成《信息表》并划转个人账户余额，转入地5个工作日内核对接收《信息表》和个人账户金额，同步到本地医保信息平台；
  4. 进度查询：参保人员/用人单位可通过医保信息平台查询业务办理进度。

- **失业保险转移**
  1. 在上海停保后，向转入地失业保险经办机构提交转移申请；
  2. 转入地受理后发送联系函，上海经办机构划转对应基金、出具缴费记录，参保缴费不满1年的，只转移关系、不划转基金；
  3. 转入地办结后，缴费年限累计合并，按转入地标准核定失业金待遇。

#### 2. 住房公积金转移接续
1. **异地转入上海**：在上海开立公积金账户并正常缴存后，通过全国住房公积金小程序、随申办APP提交异地转移接续申请，上海公积金中心受理后对接转出地完成账户合并；
2. **上海转出至异地**：在上海办理公积金账户封存后，向转入地公积金中心提交转移申请，按转入地流程办理，上海公积金中心配合完成余额划转和信息转移，全程不超过11个工作日。

  2. **贷款利率**

## 二、工伤处理

### （一）认定标准与流程

#### 1. 法定认定标准
- **应当认定为工伤的情形**
  1. 在工作时间和工作场所内，因工作原因受到事故伤害的；
  2. 工作时间前后在工作场所内，从事与工作有关的预备性或者收尾性工作受到事故伤害的；
  3. 在工作时间和工作场所内，因履行工作职责受到暴力等意外伤害的；
  4. 患职业病的；
  5. 因工外出期间，由于工作原因受到伤害或者发生事故下落不明的；
  6. 在上下班途中，受到非本人主要责任的交通事故或者城市轨道交通、客运轮渡、火车事故伤害的；
  7. 法律、行政法规规定应当认定为工伤的其他情形。

- **视同工伤的情形**
  1. 在工作时间和工作岗位，突发疾病死亡或者在48小时之内经抢救无效死亡的；
  2. 在抢险救灾等维护国家利益、公共利益活动中受到伤害的；
  3. 职工原在军队服役，因战、因公负伤致残，已取得革命伤残军人证，到用人单位后旧伤复发的。

- **不得认定/视同工伤的情形**：故意犯罪的；醉酒或者吸毒的；自残或者自杀的。

- **参保范围补充**：65周岁以下超法定退休年龄就业人员、实习生，可由用人单位单独参加工伤保险，参保后发生工伤的，按规定享受工伤保险待遇；工程建设项目流动从业人员按项目参保，纳入工伤保险保障范围。

#### 2. 认定办理流程
1. **申请时限**：用人单位需在事故伤害发生/被诊断为职业病之日起30日内，向单位注册地的区人社局提交工伤认定申请；单位未按时申请的，职工/近亲属/工会可在1年内直接提交申请。
2. **申请材料**：工伤认定申请表、劳动关系证明材料、医疗诊断证明/职业病诊断证明书、事故伤害相关证明材料。
3. **受理审核**：区人社局15日内作出受理决定，材料不全的一次性告知补正；受理后可对事故伤害进行调查核实，相关单位和个人需配合。
4. **认定决定**：自受理之日起60日内作出工伤认定决定，出具《认定工伤决定书》或《不予认定工伤决定书》，并书面送达用人单位和职工本人。
5. **劳动能力鉴定**：工伤认定后，职工伤情稳定后存在残疾、影响劳动能力的，向区、县鉴定委员会申请伤残等级、生活自理障碍等级鉴定，60日内作出鉴定结论，必要时可延长30日。

### （二）赔偿项目计算
赔偿项目分**工伤保险基金支付**和**用人单位支付**两大类，未依法参保的，全部待遇由用人单位按标准全额承担。

#### 1. 工伤保险基金支付项目及计算标准
| 赔偿项目 | 计算标准 |
| :--- | :--- |
| 工伤医疗 / 康复费用 | 符合工伤保险诊疗、药品、住院服务标准的，全额报销 |
| 住院伙食补助费 | 上海市现行标准30元/天 |
| 异地就医交通食宿费 | 经批准异地就医的，按上海市规定标准全额报销 |
| 伤残辅助器具费 | 符合国家和上海市标准的，全额报销 |
| 生活护理费 | 经鉴定确认后，按上海市上年度职工月平均工资计发：完全不能自理50%、大部分不能自理40%、部分不能自理30%；当前标准：生活完全不能自理6217元/月，生活大部分不能自理4974元/月，生活部分不能自理3731元/月 |
| 一次性伤残补助金 | 按工伤前12个月平均月缴费工资×对应月数：一级27个月、二级25个月、三级23个月、四级21个月、五级18个月、六级16个月、七级13个月、八级11个月、九级9个月、十级7个月 |
| 一到四级伤残津贴 | 按月发放，按本人工资比例计发：一级90%、二级85%、三级80%、四级75%；伤残津贴最低标准：致残一级9400元/月，致残二级8756元/月，致残三级8230元/月，致残四级7691元/月 |
| 一次性工伤医疗补助金 | 解除/终止劳动关系时发放，按上海市上年度职工月平均工资×对应月数：五级18个月、六级15个月、七级12个月、八级9个月、九级6个月、十级3个月 |
| 丧葬补助金（工亡） | 按上海市上年度职工月平均工资×6个月计发 |
| 供养亲属抚恤金（工亡） | 按职工本人工资比例计发：配偶每月40%，其他亲属每人每月30%，孤寡老人/孤儿每人每月在上述标准的基础上上浮10%，合计不高于职工生前本人工资 |
| 一次性工亡补助金（工亡） | 按上一年度全国城镇居民人均可支配收入×20倍计发 |

#### 2. 用人单位支付项目及计算标准
| 赔偿项目 | 计算标准 |
| :--- | :--- |
| 停工留薪期工资 | 停工留薪期内，原工资福利待遇不变，按月支付；一般不超过12个月，伤情严重的经批准可延长，但延长不得超过12个月 |

| 一次性伤残就业补助金 | 解除/终止劳动关系时发放，按上海市上年度职工月平均工资×对应月数：五级18个月、六级15个月、七级12个月、八级9个月、九级6个月、十级3个月 |
| 停工留薪期护理费 | 生活不能自理的工伤职工，停工留薪期内需要护理的，由用人单位负责 |

### （三）争议解决途径
1. **工伤认定争议**：职工或用人单位对工伤认定决定、不予受理决定不服的，可自收到决定书之日起60日内申请行政复议；对复议决定不服的，15日内可提起行政诉讼，也可自收到决定书之日起6个月内直接提起行政诉讼。
2. **工伤待遇支付争议**：职工与用人单位就工伤待遇、停工留薪期工资、伤残就业补助金等发生争议的，属于劳动争议，可自争议发生之日起1年内，向用人单位注册地/劳动合同履行地的区劳动人事争议仲裁委员会申请劳动仲裁；对仲裁裁决不服的，15日内可向人民法院提起民事诉讼。
3. **其他争议**：对工伤保险经办机构核定的待遇有异议的，可申请行政复议或提起行政诉讼；用人单位未依法缴纳工伤保险费的，可向上海市税务部门投诉举报，税务部门责令限期补缴。

## 三、失业保险

### （一）领取条件判断
同时满足以下全部条件，可申领上海市失业保险金：
1. 失业前用人单位和本人累计缴纳失业保险费满1年，且上海为最后缴费地；
2. 非因本人意愿中断就业（包括劳动合同期满终止、用人单位解除/辞退、用人单位违法用工导致劳动者解除劳动合同等情形）；
3. 已办理失业登记，并有求职要求。

**补充权益**：领取失业保险金期间，由失业保险基金代缴职工基本医疗保险（含生育保险）费，个人不缴费，正常享受职工医保待遇；可享受免费职业介绍、职业培训等就业服务。

**不得领取的情形**：重新就业的；应征服兵役的；移居境外的；享受基本养老保险待遇的；被判刑收监执行或者被劳动教养的；无正当理由拒不接受公共就业服务机构介绍的适当工作或培训的。

---

### （二）金额计算
#### 1. 发放标准（2025年7月1日起执行）
- 第1-12个月领取标准：2305元/月（上海市最低工资标准的90%）；
- 第13-24个月领取标准：1844元/月（按前12个月标准的80%计发）；
- 延长领取标准：领取24个月后仍未就业且符合延长条件的，按1650元/月计发。

#### 2. 领取期限核定
- 累计缴费满1年不满2年，领取期限2个月；
- 累计缴费满2年不满3年，领取期限4个月；
- 累计缴纳失业保险费年限每增加1年，期限增加2个月；
- 累计缴费满3年不满5年，领取期限最长12个月；
- 累计缴费满5年不满10年，领取期限最长18个月；
- 累计缴费满10年以上，领取期限最长24个月。

**补充规则**：重新就业后再次失业的，缴费时间重新计算，未领取完的期限可合并计算，合并后最长不超过24个月。

---

### （三）申领流程
1. **线上申领**（全程网办，无需线下材料）：通过随申办APP/小程序、“掌上12333”APP、上海市人社局官网一网通办专栏，搜索“失业保险金申领”，核对个人信息后提交申请即可。
2. **线下申领**：携带本人身份证、社保卡，至户籍所在地/居住地的区就业促进中心、街道社区事务受理服务中心，现场办理失业登记和申领手续。
3. **审核发放**：经办机构5个工作日内完成审核并告知结果；审核通过的，失业保险金自失业登记之日起计算，按月发放至申请人社保卡银行账户或指定银行账户，领取期间按月完成就业状态确认即可正常发放。
